Custom Pagebreak in Quotation Report

Video Tutorial

With this module user can easily modify Quotation/Orders Report by adding pagebreak at sale order line level.

Prerequisites before Installation

No prerequisites are required before the installation.

Configurations

No configurations.


Once you have installed the module. Go to Sales => Quotation/Sale Orders. Activate pagebreak field at sale order line level whose order line you want to display in next page of Quotation/Order Report as shown in below image

Quotation Report

As shown in above image, we have activated pagebreak field for second order line. Hence, before printing second order line pagebreak would be called and it would be displayed in the next page of report.
